How Transfer Requests Work

1. Submit Request

You select an upcoming appointment and explain why it needs to be transferred.

2. Admin Review

The admin checks whether another staff member is available for the same service and time.

3. Client Notified

If approved, the appointment is reassigned and the client receives an email update.
